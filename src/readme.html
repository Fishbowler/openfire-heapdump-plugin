Heapdump Plugin Readme

Overview

This plugin serves one primary purposes:

  • Allow a heap dump of the Openfire JVM to be generated on demand.

Installation

Copy heapdump.jar into the plugins directory of your Openfire installation. The plugin will then be automatically deployed. To upgrade to a new version, copy the new heapdump.jar file over the existing file.

Configuration

Once loaded, the plugin will add a page to the Openfire Admin Console, under Server > Server Manager > Heap Dumps. This page can be used to configure the plugin.
